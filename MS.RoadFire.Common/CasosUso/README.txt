Guía rápida para abrir los diagramas de actividad en VSCode con PlantUML

1. Instalar la extensión "PlantUML" en VSCode.
2. Instalar Graphviz (necesario para renderizar UML).
3. Abrir cualquiera de los archivos .puml en esta carpeta.
4. Presionar Alt+D (o comando "PlantUML: Preview Current Diagram") para ver el diagrama.
5. Puedes exportar como PNG, SVG o PDF desde la vista previa.
